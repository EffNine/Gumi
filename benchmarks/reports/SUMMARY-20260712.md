## 3. Comparison with Official Ornith 9B Benchmarks

Ornith 9B's official scores use **agent frameworks** with 128K-256K context, multi-turn tool access, and self-scaffolding. Novexa does not replace the agent — it improves **every turn** within the agent's loop.

| Benchmark | Official Ornith 9B | How Novexa Helps |
|---|---|---|
| **Terminal-Bench 2.1** | 43.1% | JSON repair → fewer parsing failures; instruction assist → better tool calls per turn |
| **SWE-Bench Verified** | 69.4% | Instruction assist → model follows file-edit constraints; anti-loop → fewer stuck agents |
| **NL2Repo** | 27.2% | Tool call reliability (100%) → fewer broken tool invocations |
| **ClawEval** | 63.1% | Structured output → reliable JSON for test harnesses |

> **Novexa does not compete with agent frameworks.** It improves the model layer so any agent framework (OpenCode, Continue, Claude Code, Terminus-2) gets more reliable responses per turn, which compounds into higher end-to-end success rates.

## 5. New: Instruction-Following Assist (v0.1.0-alpha)

Automatically detects 14 constraint types from user prompts and injects explicit reminders:

**Before:** Model receives raw prompt → often misses constraints
**After:** Novexa extracts constraints → adds numbered hints → validates response → retries on failure

```
Prompt: "2 sentences, end with 'learning', no word 'language'"
→ Novexa injects:
  "CRITICAL: 1. Exactly 2 sentences. 2. End with 'learning'.
   3. Do NOT use the word 'language'."
→ Valid response in 1 attempt ✅
```

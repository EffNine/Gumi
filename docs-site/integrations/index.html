Integrations

Gumi is a drop-in OpenAI-compatible endpoint. Any tool that supports OpenAI's API can use Gumi by changing the base_url and api_key.

OpenCode

Configure OpenCode to use Gumi as its LLM provider:

~/.opencode/config.json
{
  "providers": {
    "gumi": {
      "name": "Gumi",
      "apiBase": "http://127.0.0.1:8787/v1",
      "apiKey": "gumi-local",
      "models": [
        {
          "name": "lmstudio:qwen2.5-coder-7b-instruct",
          "type": "chat"
        },
        {
          "name": "lmstudio:ornith-1.0-9b@q4_k_m",
          "type": "chat"
        }
      ]
    }
  }
}

Continue

Add Gumi as a provider in Continue's config:

~/.continue/config.json
{
  "models": [
    {
      "title": "Gumi (Qwen 2.5 Coder)",
      "provider": "openai",
      "model": "lmstudio:qwen2.5-coder-7b-instruct",
      "apiBase": "http://127.0.0.1:8787/v1",
      "apiKey": "gumi-local"
    },
    {
      "title": "Gumi (Ornith 9B)",
      "provider": "openai",
      "model": "lmstudio:ornith-1.0-9b@q4_k_m",
      "apiBase": "http://127.0.0.1:8787/v1",
      "apiKey": "gumi-local"
    }
  ],
  "tabAutocompleteModel": {
    "title": "Gumi (Tab)",
    "provider": "openai",
    "model": "lmstudio:qwen2.5-coder-7b-instruct",
    "apiBase": "http://127.0.0.1:8787/v1",
    "apiKey": "gumi-local"
  }
}

Note: For tab autocomplete, you may want to use LM Studio directly for lower latency. Gumi's pipeline adds ~100-500ms overhead which can feel sluggish for inline completions.

Cline

Configure Cline to use Gumi as its API provider:

Cline Settings → API Provider
API Provider: OpenAI Compatible
Base URL: http://127.0.0.1:8787/v1
API Key: gumi-local
Model: lmstudio:qwen2.5-coder-7b-instruct

Open WebUI

Connect Open WebUI to Gumi as an OpenAI-compatible endpoint:

Open WebUI → Admin Settings → Connections
URL: http://127.0.0.1:8787/v1
Key: gumi-local

Any OpenAI SDK

All OpenAI-compatible SDKs work with Gumi by changing two values:

Python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:8787/v1",
    api_key="gumi-local"
)
Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://127.0.0.1:8787/v1',
  apiKey: 'gumi-local',
});

# Gumi Client Examples

Starter examples for connecting to Gumi's OpenAI-compatible local endpoint.

| Example | Language / Tool | Location |
|---------|-----------------|----------|
| cURL | shell | [curl/](curl/) |
| Python OpenAI | Python | [python-openai/](python-openai/) |
| LAN GPU (Mac + Windows) | ops | [lan-gpu-setup.md](lan-gpu-setup.md) |

Planned additions:

- Node.js OpenAI client
- Continue.dev config
- Cline config
- Open WebUI config

All examples assume the default Gumi URL (`http://127.0.0.1:8787/v1`) and token (`gumi-local`).

# Contributing to Novexa

Thank you for your interest in Novexa.

Novexa is a proprietary, closed-source product. The public repository is a
release and documentation mirror only.

## How you can contribute

- **Report bugs** — open a GitHub issue with steps to reproduce.
- **Suggest improvements** — open a GitHub issue with your use case.
- **Fix documentation** — submit a pull request for docs/ or README changes.
- **Share feedback** — contact **legal@effnine.com** for commercial inquiries.

Code contributions cannot be accepted because the source code is not public.
